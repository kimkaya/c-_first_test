=====================================================
   음악 파일 추가 방법 (How to Add Music Files)
=====================================================

이 폴더에 음악 파일을 넣으면 게임에서 자동으로 재생됩니다!

지원되는 파일 형식:
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)

사용 방법:
1. 이 폴더에 음악 파일을 복사합니다
2. 게임을 실행합니다
3. Start 버튼을 누르면 음악이 자동으로 재생됩니다!

참고:
- 여러 파일이 있으면 첫 번째 파일이 재생됩니다
- 무료 음악은 다음 사이트에서 다운로드할 수 있습니다:
  * FreePD.com
  * Incompetech.com
  * Bensound.com
